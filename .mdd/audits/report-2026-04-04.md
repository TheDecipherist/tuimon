## Fix Plan

### Phase 1 — Critical Fixes (HIGH severity, ~60 min)

1. **H1: Path traversal** — Use `path.resolve()` + trailing separator comparison in server.ts
2. **H2: TTY guard** — Wrap setRawMode in isTTY check in keyhandler.ts
3. **H3: Unhandled rejection** — Add .catch() to router.handleKey in index.ts
4. **H4: Circuit breaker** — Add failure counter with max retries on refresh interval
5. **H5: Sixel OOM** — Stream sixel output in chunks rather than building full string
6. **H6: Color overflow** — Log warning when palette exceeds 256 colors
7. **H7: F-key await** — Await F-key action in handleKey rather than fire-and-forget

### Phase 2 — Medium Fixes (~85 min)

8. **M2: URIError** — Wrap decodeURIComponent in try/catch
9. **M7+M8: Router cleanup** — Fix non-null assertion, remove dead ternary
10. **M9: stdin.pause** — Call pause() in keyhandler stop()
11. **M10: Dead code** — Remove unused `plain` variable in fkeybar
12. **M12: SIGINT** — Await stop() in SIGINT handler
13. **M13: Render mutex** — Add simple lock around renderFrame
14. **M14+M15: Assertions** — Fix empty pages guard, add unhandledRejection to CLI

### Phase 3 — LOW severity items (deferred)

Address as encountered in normal development.
